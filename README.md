## QA

- 为什么不用 TypeScript 写？
  面试平台的在线 IDE 大部分都是只支持 JavaScript

- 为什么不用 import?
  大部分的沙盒支持到 Node.js 10.x 版本，要支持 import 写麻烦

- 为什么两格缩进?
  VS Code 设置了这个
